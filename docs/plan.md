# Plan

## Planirani koraci u praktičnom dijelu

**Korak 1: Postavljanje laboratorijskog okruženja**

**Korak 2: Identifikacija servisa i ranjivosti**

**Korak 3: Eksploatacija i dobivanje TCP ljuske**

**Korak 4: Implementirati C2 poslužitelj**

**Korak 5: Upravljanje botovima preko C2**

**Korak 6: Simulacija DDoS napada**

**Korak 7: Snimanje i analiza mrežnog prometa**

**Korak 8: Otkrivanje i mjere obrane**

**Korak 9: Izrada dokumentacija i završno izvješće**

## Planirani alati

- **Docker**: izoliranih komponenti i mreže.
- **Metasploit Framework**: automatizirana eksploatacija ranjivosti.
- **Nmap**: identifikacija portova, servisa i verzija.
- **Flask (Python)**: jednostavan C2 HTTP API.
- **curl**: dohvat naredbi/payload-a i generiranje HTTP prometa.
- **tcpdump**: snimanje prometa u `.pcap`.
- **Wireshark**: analiza snimljenog prometa i vizualizacija napada.
